KeyValue

Allows apps to persist json values associated to a project and a key.

// example
const user = {
  firstName: 'John',
  lastName: 'Doe'
};
const keyValue = new KeyValue();
keyValue.set('testApp', 'profile', user);
const profile = keyValue.get('testApp', 'profile');
